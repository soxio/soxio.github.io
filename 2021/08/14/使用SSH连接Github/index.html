使用SSH连接Github

2021-08-14
Github-gitgithub

本地生成密钥SSH key

ssh-keygen -t rsa -C “用户邮箱”

查看生成的密钥

cat ~/.ssh/id_rsa.pub

Github设置密钥

复制上面查看的key，打开github，在头像下面点击settings，再点击SSH and GPG keys，新建一个SSH，名字随便。

验证是否连接成功

ssh -T git@github.com

觉得不错，就打赏吧.
