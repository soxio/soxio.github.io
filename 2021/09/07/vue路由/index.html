vue路由的基本使用

2021-09-07
869- 4m- -
vue-vue路由

安装

npm install vue-router

配置

index.js

  • 在src目录下新建router文件夹，并新建index.js
  • 导入对应的包并挂载路由
// 1. 导入包
import Vue from 'vue'
import VueRouter from 'vue-router'
// 2. 挂载路由
Vue.use(VueRouter)

  • 导入需要路由的组件（两种方式）

    // 1. 直接导入组件
import demo1 from ''
// 2. 路由懒加载(推荐) 当打包构建应用时，JavaScript 包会变得非常大，影响页面加载。如果我们能把不同路由对应的组件分割成不同的代码块，然后当路由被访问的时候才加载对应组件，这样就更加高效了。
const demo1 = ()=> import('../components/demo1.vue')

  • 创建 路由——组件 映射关系

    const routes = [
  {
    path: '/demo1',
    component: demo1
  }
 ]

  • 创建路由对象并导出

    const router = new VueRouter({
  // 在路由对象中应用上一步的路由信息
  routes,
  // 更改路由模式
  mode: 'history'
})
// 导出路由信息
export default router

main.js

  • 导入路由文件 index.js

    import router from './router/index'

  • 挂载路由

    new Vue({
  // 添加路由对象
  router,
  render: h => h(App),
}).$mount('#app')

App.vue

  • 创建路由元素连接

    <!-- to="路由路径"		tag="元素类型 -->
<router-link to='/demo1' tag="button">demo1</router-link>

  • 展示路由对应组件页面

    <!-- 直接展示 -->
<router-view></router-view>

路由跳转

方式一：页面元素跳转

<router-link to="/find">发现音乐</router-link>

方式二：js代码跳转

  • 路由无需传值时直接push路径

    toMy() {
      this.$router.push('路由路径')
 }

  • 通过path属性来进行跳转

    传参：只能通过query进行传参

    toMy() {
      this.$router.push({
        path: '/myMusic',
        query: {
          name: '张三'
        }
      })
}

  • 通过name属性来进行跳转

    注意：params传值只能和name属性结合使用

    tofriend() {
      this.$router.push({
        // params 传参需要路由有配置name
        name: 'tofriend',
        params: {
          name: '李四'
        }
      })
}

    重定向

    const routes = [
  {
    path: '/',
    redirect: '要重定向的路由'
  }
 ]

    添加 404 not found

    const routes = [
  {
    path: '*',
    component: notfound
  }
 ]

    路由传值

方式一

  • 通过在路由后面直接跟上问号键值对的方式

    App.vue

    <router-link to="/myMusic?name=张三">我的音乐</router-link>

  • 通过 this.$route.query.name 取值，插值无需跟上this

    myMusic.vue

    <script>
export default {
  created() {
    console.log(this.$route.query.name)
  }
}
</script>

方式二

  • 通过在路由配置路径 / 方式传值

    App.vue

    <router-link to="/friend/李四">朋友</router-link>

    路由文件 index.js

    const routes = [
	{
    // 参数必须
    // path: '/friend/:name',
    // 加了问号表示参数可有可无
    path: '/friend/:name?',
    component: friend,
  }
]

    firend.vue

  • 通过 this.$route.params.name 取值，插值无需跟上this

    <script>
export default {
  created() {
    console.log(this.$route.params.name)
  }
}
</script>

路由守卫

添加全局守卫

main.js

let isLogin = false;

// 全局守卫 路由跳转前判断
router.beforeEach((to, from, next) => {
  console.log(to, from, next);
  if (to.path == '/myMusic' && isLogin == false) {
    alert('请登录');
    next(false);
  } else {
    next()
  }
})
